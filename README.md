# Portfolio Website

## Overview
A personal portfolio website built with React to showcase my web development projects, technical skills, and experience. The website contains detailed information about the applications I have created, including the technologies used and the features implemented.

It also includes links to my CV, GitHub, and LinkedIn profiles, allowing potential employers to learn more about my background and experience.

## Features

- Responsive design for desktop and mobile devices.
- Projects section displaying my portfolio pieces and technologies used.
- Technical skills section highlighting my experience with React, TypeScript, Node.js, Express, MySQL, HTML, CSS, and TailwindCSS.
- About me section with a summary of my experience and interests in web development.
- Links to my CV, GitHub, and LinkedIn profiles.
- Smooth navigation between sections/pages.
- Contact section for potential employers.
